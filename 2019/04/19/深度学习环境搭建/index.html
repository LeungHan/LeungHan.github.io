记录在win10操作系统下安装Ubuntu双系统并搭建深度学习环境

深度学习环境搭建

一、Linux双系统安装

1.1 工具准备

  1. Linux IOS镜像文件
  2. UltraISO（软碟通 光盘映像ISO文件编辑制作工具）
  3. U盘一个（请提前备份U盘里面的资料，因为后面操作要格式化U盘）

1.2 相关设置

1.2.1 磁盘分区

我们需要创建一个新的磁盘分区作为Ubuntu的磁盘空间，步骤如下：

  • 按Win+X组合键，然后选择“磁盘管理”，或者右键“我的电脑”->“管理”->“磁盘管理”：
  • 选择一个剩余空间足够大的磁盘，右键->”压缩卷”，然后分出新的磁盘空间（建议100G以上）

磁盘压缩完成之后就可以看到后面出现了一块黑色或绿色的未分配空间，这样磁盘分区工作就完成了

1.2.2 禁止快速启动

按住Win + X，选择“电源选项”，依次执行：

  • 选择电源按钮的功能
  • 更改当前不可用的设置
  • 取消选择”启用快速启动(推荐)”

注： “快速启动”是Windows 8时代引进的新特性，建议关闭该特性的原因是，“快速启动”会影响Grub开机引导过程，可能出现无法载入Ubuntu的状况，最后选择“保存修改”。

1.2.3 禁用安全启动(Secure Boot)

重启电脑后通过快捷键进入bios（快捷键自行百度或问京东客服），进入BIOS后一般如下图所示:(大致和下图一样)

  • 通过键盘控制光标移到Security上（不同电脑可能不一样，自行查找），回车进入
  • 选择Secure Boot，将其设置为Disable，F10保存重启

1.3 制作Ubuntu 16.04 LTS启动盘

1.3.1 解压

将IOS文件直接解压到空U盘中即可，当然不放心的话也可以使用UltraISO软碟通（试用即可）制作启动盘，实际上两者没差。具体文件如下，可以看到其中包含EFI文件夹：

如果解压不行通过UltraISO（软碟通）软件制作启动盘。

1.3.2 UltraISO（软碟通）制作

  • 打开UltraISO软件，点击 文件 -> 打开 -> 选择你的Ubuntu16.04所在的路径
  • 点击 启动 -> 写入硬盘映像 -> 选择你的U盘 -> 写入

写入完成，U盘启动盘便已经制作完毕了

1.4 安装Ubuntu 16.04 LTS系统

  • 插入U盘，重启电脑，进入boot启动选项，选择U盘启动（具体方法百度，不同电脑可能不一样）
  • 进入Ubuntu的安装界面，选择语言，然后选择安装Ubuntu
  • 接下来有两个地方可以勾选，直接不勾选
  • 进行系统分区，我们这里选择“其他选项”，自己手动为ubuntu进行分区划分
  • 双击这一部分，即可出现划分选项。或者单击“未分配”磁盘，点击＋加号，如下图所示（这块空间就是1.2.1小结得到的未分配空间，核对磁盘大小是否一致，其他磁盘为win下的磁盘空间，不要动）
  • 然后会出现分区选项，按照下文四个分区进行配置，大小不要按照图片上的进行分配

注意：Linux一共有四个最基本的分区

  1. swap交换空间：相当于Win中的虚拟内存，通常需要划分对应物理内存2倍的空间，考虑到深度学习主机内存一般都是32G、64G或者128G，所以选择忽略不划分，之后如有需要还可以在系统设置中添加swap部分，可以选择分配16G。
  2. EFI系统分区：选择分区类型为“逻辑分区”，分区位置为“空间起始位置”，分配大小为512M，足矣。
  3. 挂载“/”：为EXT4日志文件系统，选择“逻辑分区”和“空间起始位置”。根目录将挂载除了“/home”和“/usr”之外的其他目录，分配50G。
  4. 挂载“/home”：为EXT4日志文件系统，选择“逻辑分区”和“空间起始位置”。剩余空间全部分配给“/home”。

最后引导器的设备选择上文分配的EFI分区，确认无误之后，就可以点击“继续安装”，接下来就是等待系统安装进度的完成，接下来便是最后的一些小设置了，分别是设置时区，键盘布局，用户信息，设置完成之后，Ubuntu系统就已经安装完毕了，点击“现在重启”。重启之后即会出现EFI登录选项，可以选择Ubuntu或者Win10。如下图所示，如果没有出现的话，尝试在BIOS中将Ubuntu EFI作为第一启动项。至此，双系统安装成功。
